# Portfolio Website

Modern one-page portfolio with animated sections (Hero, About, Skills, Projects, Experience, Certificates, Resume, Contact).

## Tech

- React + Vite
- Tailwind CSS
- Framer Motion
- React Router

## Run Locally

```sh
npm install
npm run dev
```

## Build

```sh
npm run build
npm run preview
```

## Structure

```text
src/components/   UI sections
src/hooks/        custom hooks
src/pages/        routes
public/           static assets
```
